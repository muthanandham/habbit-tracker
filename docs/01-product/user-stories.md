# User Stories — Unified Daily Tracker (Life-OS)

> **Version:** 2.0 | **Last updated:** March 2026

## Epic 1: Onboarding & Auth

| ID | Story | Acceptance Criteria | Priority |
|---|---|---|---|
| US-001 | As a user, I want to sign up with email/password so I can securely access my Life-OS | User can create account and log in successfully | P0 |
| US-002 | As a user, I want to log in quickly on return visits | Session persists and allows direct dashboard access | P0 |

## Epic 2: Habit Setup & Management

| ID | Story | Acceptance Criteria | Priority |
|---|---|---|---|
| US-010 | As a user, I want to create a custom habit in under 30 seconds | Habit can be created with name + frequency + type | P0 |
| US-011 | As a user, I want starter templates so I can begin fast | User can add 1+ habits from template packs | P1 |
| US-012 | As a user, I want category tags to organize habits | Habits can be filtered by category (Focus, Wellness, etc.) | P1 |
| US-013 | As a user, I want to archive habits I pause | Archived habits disappear from today checklist | P0 |

## Epic 3: Task Management (Productivity)

| ID | Story | Acceptance Criteria | Priority |
|---|---|---|---|
| US-020 | As a user, I want to add "Must-Do" tasks for today | Task can be added with just a title | P0 |
| US-021 | As a user, I want a single checklist for habits and tasks | Both appear in a unified "Today" view | P0 |
| US-022 | As a user, I want unfinished tasks to carry over automatically | Tasks not completed today appear on tomorrow's list | P1 |

## Epic 4: Wellness & Reflection

| ID | Story | Acceptance Criteria | Priority |
|---|---|---|---|
| US-030 | As a user, I want to log my mood and energy daily | 1-5 scale or emoji-based quick selection | P1 |
| US-031 | As a user, I want to track my hydration easily | Quick-tap for a "glass of water" or volume entry | P1 |
| US-032 | As a user, I want a daily reflection/journal space | A simple text area for EOD thoughts | P1 |

## Epic 5: Unified AI Assistant

| ID | Story | Acceptance Criteria | Priority |
|---|---|---|---|
| US-040 | As a user, I want to log activities via natural language | Chat bar parses "Drank 2 glasses of water" or "Did 10 pushups" | P0 |
| US-041 | As a user, I want the AI to summarize my day | AI provides a concise "Wrap-up" at the end of the day | P1 |
| US-042 | As a user, I want proactive nudges from the assistant | Assistant sends in-app nudges if mood is low or water is low | P2 |

## Epic 6: Progress, Streaks & Insights

| ID | Story | Acceptance Criteria | Priority |
|---|---|---|---|
| US-050 | As a user, I want to see my current/best habit streaks | Visual streak counter on habit cards | P0 |
| US-051 | As a user, I want weekly completion percentage | Dashboard metric for overall consistency | P0 |
| US-052 | As a user, I want to see correlations in my data | Insight card shows "Consistency increases when Sleep > 7h" | P2 |

## Epic 7: Reminder & Notification Center

| ID | Story | Acceptance Criteria | Priority |
|---|---|---|---|
| US-060 | As a user, I want in-app reminders based on time-phased UI | Morning/Night notifications depending on module focus | P1 |
| US-061 | As a user, I want to snooze or dismiss reminders | One-tap snooze or dismiss action | P1 |
