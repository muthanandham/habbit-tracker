# Privacy Compliance
## Habbit Tracker — Privacy Baseline

> **Version:** 1.0 | **Last updated:** March 2026

## Data We Store

- Account: email, password hash
- Habits and check-ins
- Optional AI setup inputs (limited retention)

## Data We Do Not Store

- Payment data
- Government identity documents
- Unnecessary device identifiers

## User Controls

- Edit profile and timezone
- Delete habit history (scoped)
- Delete account (full removal workflow)

## Retention Baseline

- Active account data retained while account is active
- AI logs retained short-term for reliability/debug only

## Privacy Commitments (v1)

- No third-party ad trackers by default
- Minimal analytics footprint
- Clear privacy summary in settings
