# Cost & Usage Policy
## Habbit Tracker — v1 Cost Controls

> **Version:** 1.0 | **Last updated:** March 2026

## Cost Objectives

- Keep monthly infrastructure spend predictable for solo operation
- Avoid uncontrolled AI usage costs

## Infrastructure Budget (Target)

- Frontend hosting: low/free tier
- Backend hosting: starter tier
- Database: MongoDB Atlas entry tier
- Monitoring: free/low tier tools

## AI Usage Controls

- Per-user daily quota on AI setup calls
- Global daily cap for AI provider
- Template fallback when quota exceeded

## Cost Guardrails

- Alert at 70%, 90%, and 100% monthly budget thresholds
- Disable non-essential AI endpoints if budget is exceeded
- Prioritize core tracking uptime over AI features
