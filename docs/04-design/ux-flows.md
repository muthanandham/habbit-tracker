# UX Flows
## Unified Daily Tracker (Life-OS) — User Journeys

> **Version:** 2.0 | **Last updated:** March 2026

## Primary Navigation Flow

1. User signs in
2. Lands on **Mission Control** (The Unified Today Dashboard)
3. Navigates to:
   - **Assistant:** AI Chat entry
   - **Board:** Deep module-level views
   - **Insights:** Weekly stats & correlations
   - **Settings:** Profile & preferences

## FLOW-A: Unified First Sign-Up

1. **Account Creation:** Secure email/password auth.
2. **Preference Match:** User chooses "Focus Pillars" (e.g., Focus on Wellness + Productivity).
3. **The "Guided Log":** AI walks the user through their first Day Log (Habits, Tasks, and current Mood).
4. **Dashboard Setup:** System pre-populates the dashboard with a "Starter Pack."

## FLOW-B: The "Unified Log" (AI-First)

1. **Entry:** Tap floating chat icon or type in command bar.
2. **Input:** "Did 20 mins of yoga and finish the report. Energy is 4/5."
3. **Parsing:** AI identifies 1 Habit check-in, 1 Task update, and 1 Wellness log.
4. **Confirmation:** System shows "Summary Card" of identified actions.
5. **Sync:** User confirms (or auto-confirms); Dashboard updates instantly.

## FLOW-C: Morning Planning (Goal Focus)

1. **Nudge:** Assistant sends "Morning Briefing" (Daily goals + Pending tasks).
2. **Review:** User views "Must-Do" task list.
3. **Commencement:** User picks a task to "Focus On" (Minimalist Focus Mode UI).

## FLOW-D: Daily Habit Tracking & Maintenance

1. **Dashboard Taps:** User interacts with "Quick-Tap" habit icons (Binary/Numeric/Timer).
2. **Success Feedback:** Every tap triggers a "Glow" effect; Streak count increment visible.
3. **Undo:** 5-second "Mistake" button available for all manual/AI logs.

## FLOW-E: Evening Reflection & Wellness

1. **Prompt:** "Ready to wind down? How was your day overall?"
2. **Journaling:** User types or speaks a quick reflection.
3. **Mood Check:** Log energy and mood for the end-of-day record.
4. **Summary:** Assistant provides an "Achievements" recap for the day.

## FLOW-F: Finding Insights & Correlations

1. **View:** Open "Insights" module.
2. **Exploration:** View "Cross-Module Trends" (e.g., "Mood Trends vs. Task Completion").
3. **AI Recommendation:** Assistant suggests a minor routine tweak based on patterns.
